## 문제 요약
- N개의 용액이 주어짐 (2 <= N <= 100000)
- 두 용액을 섞어 특성값을 0에 최대한 가깝게 만드는 두 용액을 구하는 것이 목적.

## 입출력 예시
- 입력
```angular2html
5
-99 -2 -1 4 98
```
- 출력
```angular2html
-99 98
```

## 문제 유형
- 투 포인터
- 이진탐색

## 핵심 아이디어
- 원소를 순회하면서 원소를 기준으로 양 옆 배열 이진탐색
- 원소를 x라고 하면 -x와 가장 가까운 원소를 찾는다.
- 찾은 원소와 원래 원소를 섞어서 0과 가까운 두 용액 값 갱신

## 복잡도
- O(N log N)

## 배운 점
- 이진탐색을 할 때 양 옆의 배열을 찾을 필요가 없다. 현재 원소의 부호와 반대되는 원소들이 포함된 배열 하나만 찾으면 된다. 
- 양 옆에서 투 포인터로 좁혀가면서 답을 찾아도 된다.
