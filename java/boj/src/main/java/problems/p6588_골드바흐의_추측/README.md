## 문제 요약
최대 100000개의 테스트케이스가 들어옴
1000000 이하의 값 n(n은 짝수)에 대하여 골드바흐의 추측을 만족하는 두 홀수를 구하는 것이 목적

## 입출력 예시
- 입력
```markdown
8
20
42
0
```
- 출력
```markdown
8
20
42
0
```

## 문제 유형
- 수학
- 정수론
- 소수 판정
- 에라토스테네스의 체

## 핵심 아이디어
에라토스테네스의 체로 빠르게 소수의 목록을 구한다.
두 홀수 중 한 개가 대부분 작은 수로 이루어져 있다.

## 복잡도
O(T*(log(logN) + N))
T: 테스트케이스 수
N: 입력 정수

## 배운 점
System.out.println()으로 출력하는 것이 상당한 시간이 소요된다. Stringbuilder를 이용하여 
문자열을 만드는 것이 빠르다.
그리고 한 줄의 문자열을 다 만들어서 붙이기보다는 append를 여러번 사용하여 Stringbuilder를 만드는 것이
더 빠르다. 

에라토스테네스 체를 구할 때 순회하는 값의 제곱부터 시작하면 더 최적화된 시간 복잡도를 얻을 수 있다.
